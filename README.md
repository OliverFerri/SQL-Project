### 1. Top Paying Data Analyst Jobs

To identify the highest-paying roles, I filtered positions with Tableau in their job title by average hourly salary and location, focusing on remote jobs. This query highlights the highest paying opportunitites in the field.

```sql
SELECT
    DENSE_RANK() OVER (ORDER BY salary_hour_avg DESC) AS rnk,
    job_id,
    job_title,
    job_location,
    job_schedule_type,
    salary_hour_avg,
    job_title_short,
    f.company_id,
    CAST(job_posted_date AS DATE) AS job_posted_date
FROM job_postings_fact f
JOIN company_dim c ON f.company_id = c.company_id
WHERE
    job_title LIKE '%tableau%'
    AND salary_hour_avg IS NOT NULL
    AND job_location = 'Anywhere'
LIMIT 12;
```

Here is a break down of the results:

- **All roles are fully remote** — With a strong emphasis on contractor positions (9 out of 12), indicating high demand for flexible, specialized Tableau talent.

- **Hourly rates are premium** — Top roles pay $65–$89/hour (roughly $135k–$185k annualized), showing that strong Tableau expertise combined with data analysis or development skills commands excellent compensation.

- **Job titles are diverse** — mixing Data Analyst, Tableau Developer, Data Engineer, and even Data Scientist roles, highlighting that Tableau skills are valued across multiple job families, especially in visualization and dashboard development.

### 2. Text

To identify the highest paying job skills, I filtered Data Analysts jobs that had a salary (salary was not null), focusing on remote jobs.

```sql
WITH top_jobs AS (
    SELECT
        DENSE_RANK() OVER (ORDER BY salary_year_avg DESC) AS rnk,
        job_id,
        job_title,
        salary_year_avg,
        name AS company_name
    FROM job_postings_fact j
    JOIN company_dim c ON j.company_id = c.company_id
    WHERE
        job_title_short = 'Data Analyst'
        AND salary_year_avg IS NOT NULL
        AND job_location = 'Anywhere'
)

SELECT
    t.job_id,
    job_title,
    company_name,
    salary_year_avg,
    skills
FROM top_jobs t
JOIN skills_job_dim sj ON t.job_id = sj.job_id
JOIN skills_dim s ON sj.skill_id = s.skill_id
WHERE rnk <= 12;
```

1. **Foundational / Must-Haves:** SQL + Python dominate. Nearly every role expects SQL proficiency,
   and Python is close behind. This reflects the shift toward programmable,
   scalable data analysis.

2. **Visualization & BI Tools:** Tableau leads strongly, followed by Power BI and Excel.
   These are essential for communicating insights.

3. **Cloud & Data Platforms:**
   Azure, AWS, Snowflake, and Databricks appear frequently in higher-paying roles,
   indicating demand for cloud-native skills.

4. **Programming & Stats:** R remains relevant (especially in certain industries),
   alongside Python data libraries like Pandas and NumPy.

5. **DevOps / Collaboration Tools:** Higher-level or principal roles often list Git-related tools, Jira,
   Confluence, Bitbucket, etc., showing the need for version control and team workflows.

### 3. Text

To find the skill with the most demand, I looked at data analyst jobs that were either remote or located in dallas.

```sql
SELECT skills, COUNT(f.job_id) AS skill_count
FROM job_postings_fact f
JOIN skills_job_dim sj ON f.job_id = sj.job_id
JOIN skills_dim s ON sj.skill_id = s.skill_id
WHERE
    job_title_short = 'Data Analyst'
    AND (job_location LIKE '%Dallas%' OR job_location = 'Anywhere')
GROUP BY skills
ORDER BY skill_count DESC
LIMIT 5;
```

### 4. Text

To identify the top skills based on salary, I aggregated the data on the skill to find the average salary for that skill. I focused on data analyst jobs that had a salary (not null) and were either remote or located in Dallas.

```sql
SELECT skills, ROUND(AVG(salary_year_avg), 0) AS avg_salary
FROM job_postings_fact f
JOIN skills_job_dim sj ON f.job_id = sj.job_id
JOIN skills_dim s ON sj.skill_id = s.skill_id
WHERE
    job_title_short = 'Data Analyst'
    AND salary_year_avg IS NOT NULL
    AND (job_location LIKE '%Dallas%' OR job_location = 'Anywhere')
GROUP BY skills
ORDER BY avg_salary DESC
LIMIT 25;
```

Summary of Trends:

1. Highest rewards go to Big Data tools and specialized platforms.

2. There’s a clear blurring of lines between Data Analyst and Data Engineer roles — the best-paid “analyst” jobs require pipeline, cloud, and dev skills.

3. Python + Cloud + Big Data stack appears to be the golden combination for top compensation.

4. Niche tools can give big salary boosts in specific industries/companies.
